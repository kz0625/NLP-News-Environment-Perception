# NLP-News-Environment-Perception
本项目实现了一个基于新闻环境感知（NEP）的假新闻检测系统。核心创新在于将新闻内容与其所处的宏观环境（近期的整体新闻趋势）和微观环境（语义最相关的新闻集合）结合进行综合判断，突破了传统方法仅聚焦于文本本身或依赖社交网络的局限。  项目构建了完整的数据处理流水线，利用SimCSE模型生成文本语义向量，并通过时间窗口和相似度筛选构建新闻环境。采用高斯核池化提取环境感知特征，设计门控机制融合宏观与微观特征，最终与BERT、EANN等基础检测器结合进行分类。  在新数据集（gossip数据集）上的实验表明，该框架能有效提升检测性能，验证了新闻环境特征对于识别语义偏离的假新闻具有重要价值，为假新闻检测提供了新的多维度分析视角。
